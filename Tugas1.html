1. apa itu version control?

Version control adalah sistem yang mencatat setiap perubahan pada satu atau sekumpulan berkas dari waktu ke waktu. Dengan sistem ini, kita bisa dengan mudah memantau riwayat perubahan dan kembali ke versi spesifik kapan pun dibutuhkan.

2. Perbedaan Git dan Github

Git adalah salah satu sistem pengontrol (Version Control). Tool yang sering digunakan dalam proyek pengembangan software sedangkan GitHub merupakan manajemen project, sistem versioning code, sekaligus platform jaringan sosial bagi para developer seluruh dunia. GitHub memberikan layanan cloud untuk menyimpan dan mengelola project/repository git.

3. Apa itu HTML?

HyperText Markup Language adalah standard markup language yang digunakan untuk membuat dan menyusun halaman web. Halaman web disusun dari beberapa tulisan yang berisi heading, paragraf, link, gambar dan atribut lain di dalam dokumen HTML.
